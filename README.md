# Spring Cloud 알아보기

### 0. 개요
- Spring Cloud가 제공하는 서비스를 이용하여 Micro Service Architecture를 구성하는 방법에 대해 실습합니다.
- daddyprogrammer.org에서 연재 및 소스 Github 등록
    - https://daddyprogrammer.org/post/series/springcloud/

- Java 8~11
- SpringBoot 2.x
- Spring Cloud
- Intellij Community
